# odev

CLI para gestion de entornos de desarrollo Odoo.

## Instalacion

```bash
pip install odev
```

## Uso rapido

```bash
# Crear un nuevo proyecto
odev init mi-proyecto

# Levantar el entorno
cd mi-proyecto
odev up

# Ver estado de los servicios
odev status
```

## Requisitos

- Python 3.10+
- Docker con Docker Compose v2
